# NodeJS API Stater Template (ExpressJS)

This is a NodeJS API Stater Template with ExpressJS Framework

1. Run `npm install`
2. Run `cp .env.example .env`
3. Configure database credential on the `.env` file
4. Run `npm start` to start server
